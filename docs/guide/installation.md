# 安装

### 版本

I-UI 目前还处于快速开发迭代中 v0.1.0

### 使用包管理器

建议使用npm/yarn/pnpm安装 I-UI

::: code-group

```shell [npm]
npm install mini-i-ui
```

```shell [Yarm]
yarn install mini-i-ui

```

```shell [Pnpm]
pnpm install mini-i-ui

```

:::

如果你的网络环境不佳，推荐使用[cnpm](https://github.com/cnpm/cnpm)

```shell
npm config set registry https://registry.npmmirror.com
```
